# 启动说明

## 首次运行

```bash
# 启动基础设施 (Nacos + MySQL + Redis + ES + RabbitMQ + MinIO + 可观测性)
cd docker && bash setup.sh
```

## 后端微服务（按顺序启动，各开一个终端）

Nacos 启动后，各服务会自动注册发现，顺序不重要，建议按以下顺序：

```bash
# 1. 网关
cd gateway-service && mvn spring-boot:run

# 2. 基础服务
cd user-service && mvn spring-boot:run
cd oss-service  && mvn spring-boot:run

# 3. 业务服务
cd forum-service && mvn spring-boot:run
cd notification-service && mvn spring-boot:run
cd announcement-service && mvn spring-boot:run

# 4. AI 服务（需要 DEEPSEEK_KEY 环境变量）
cd ai-service && mvn spring-boot:run
```

API 统一入口: `http://localhost:8081` (网关)

## 前端

```bash
cd my-project-frontend && npm run dev
```

访问: `http://localhost:5173`（前端 dev server）
