# 启动说明

## 0. 配置 API 密钥（首次运行前）

确认以下 `.env.properties` 文件已存在（密钥从 `my-project-backend/.env.properties` 迁移）：

```bash
ai-service/.env.properties        # DEEPSEEK_KEY, TAVILY_API_KEY 等
forum-service/.env.properties     # WEATHER_KEY
user-service/.env.properties      # STMP_EMAIL, STMP_EMAIL_PASSWORD, JWT_KEY
```

## 1. 启动基础设施

```cmd
cd docker
docker compose up -d
```

确认 Nacos 控制台：http://localhost:8848

## 2. 启动后端微服务

### 一键启动（推荐）

双击或运行：

```cmd
start-all.bat
```

这会为每个服务打开一个独立的 cmd 窗口，依次启动。关闭时运行：

```cmd
stop-all.bat
```

### 手动逐个启动

各开一个终端，依次执行：

```cmd
cd gateway-service       && mvn spring-boot:run    @ 端口 8081
cd user-service          && mvn spring-boot:run    @ 端口 8082
cd oss-service           && mvn spring-boot:run    @ 端口 8084
cd forum-service         && mvn spring-boot:run    @ 端口 8088
cd notification-service  && mvn spring-boot:run    @ 端口 8085
cd announcement-service  && mvn spring-boot:run    @ 端口 8086
cd ai-service            && mvn spring-boot:run    @ 端口 8083
```

API 统一入口: `http://localhost:8081`

## 3. 启动前端

```cmd
cd my-project-frontend && npm run dev
```

访问: `http://localhost:5173`
